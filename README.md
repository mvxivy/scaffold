# Scaffold

### install tool 
```sh
npm install -D @mvxiv/scaffold
```

### Scaffold vue component

```sh
$ npx scaffold ComponentName
```

that makes this structure in your project directory
```
─src──┐
───components
─────ComponentName──┐
───────ComponentName.vue
───────ComponentName.spec.ts
───────ComponentName.scss
```
with some boilerplate content
